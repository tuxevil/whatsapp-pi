# Menu Contract Update: Blocked Numbers

## Main Menu (`/whatsapp`)
1. Login / Reconnect
2. Connect / Disconnect Agent
3. Logoff
4. Allow Numbers
5. **Blocked Numbers** (New)

## Blocked Numbers Menu
- **List Item**: `[Number]` (E.164)
- **Selection Action**: Opens Confirmation.
- **Confirmation Options**:
    - **Unblock and Allow**: Moves number to Allowed List.
    - **Delete**: Permanently removes from Block List (remains unknown).
    - **Cancel**: Returns to list.
