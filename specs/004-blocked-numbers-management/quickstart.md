# Quickstart: Blocked Numbers (Ignored Numbers)

## Understanding Blocked Numbers

**Blocked Numbers** shows all phone numbers that have tried to contact you but are **not in your Allow List**. These numbers are automatically tracked when they send messages.

## Viewing Ignored Numbers

1. Type `/whatsapp`
2. Select **Blocked Numbers**
3. See all numbers that tried to contact you but were ignored
4. Numbers are displayed with their WhatsApp name if available: "Name (number)"

## Adding an Ignored Number to Allow List

When someone you want to talk to appears in Blocked Numbers:

1. Type `/whatsapp`
2. Select **Blocked Numbers**
3. Select the contact you want to allow
4. Select **Allow**
5. Confirm the action
6. The contact is moved to **Allowed Numbers** and Pi will now respond to them

## How It Works

- **Automatic Tracking**: When a non-allowed number sends a message, it's automatically added to the ignored list
- **WhatsApp Names**: The system captures the sender's WhatsApp display name automatically
- **One-Click Allow**: Simply select a number from Blocked Numbers and click "Allow" to start interacting
- **Clean List**: Once allowed, the number is removed from the ignored list

## Managing Allow List

To manually add numbers before they contact you:

1. Type `/whatsapp`
2. Select **Allow Numbers**
3. Select **Add Number**
4. Enter the phone number (e.g., `+5511999999999`)
5. Optionally enter a name
6. The number can now interact with Pi
